# Сюда — экспортированные из Figma иконки (SVG)

Положи сюда SVG-файлы иконок, выгруженные из Figma (страница **Icon**).

## Как экспортировать из Figma

1. Открой страницу **Icon**.
2. Выдели фрейм `icons` (или все иконки-компоненты сразу).
3. Правая панель → **Export** → формат **SVG** → **Export**.
4. Сохрани файлы в эту папку (`design-system/icons/svg/`).

## Про имена

Не нужно вручную переименовывать — клади как выгрузилось
(`icon/hamburger` → `icon-hamburger.svg` и т.п.). Я сопоставлю их
с именами реестра (`hamburger`, `search`, `edit`, …) при встраивании.

## Что дальше

После того как файлы здесь появятся, я:
1. Встрою SVG-пути прямо в `design-system/icons/icons.js` (инлайн, чтобы
   работало и при открытии каталога как локального файла `file://`).
2. Переведу на реестр иконки, которые сейчас захардкожены lucide-кодом
   (nav-rail, header, статусы, алерты, таблица), чтобы и они стали реальными.

Lucide останется фолбэком для иконок, которых здесь не окажется.
